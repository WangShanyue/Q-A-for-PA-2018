# Lab2 Bomblab 授课内容

## 工作描述

每个阶段输入正确的输入，拆除每个阶段的炸弹

## 测试过程

对于每一个关卡，执行下列步骤：
1. 使用`objdump`工具查反汇编
2. 读懂反汇编代码
3. 同时使用`gdb`工具对炸弹进行跟踪调试（用一下GDB给大家看）
4. 根据读懂的反汇编代码和调试情况求得程序期待你输入的数据
5. 直接运行程序成功拆除本关卡or爆炸

## 一点心得技巧

1. 进入GDB后的第一件事情，在引爆炸弹的函数下断点，以免造成人身安全事故（后期可能的话，我们有可能上线排行榜功能）
2. 将得到的反汇编代码划片来看（以`jmp`或`jcc`为界分区域），这样方便看清代码逻辑
